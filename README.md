## Create a .env file in the server/ directory:
- **PORT=5000**
- **MONGO_URI=your_mongodb_connection_string**
- **JWT_SECRET=your_jwt_secret**
- **EMAIL_USER=your_email@example.com**
- **EMAIL_PASS=your_email_password**
